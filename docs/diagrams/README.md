# Diagrammes UML - MedIQ

Ces fichiers sont prepares pour le rapport de projet :

- `use-case-medicalvision.puml` : diagramme de cas d'utilisation global.
- `class-diagram-medicalvision.puml` : diagramme de classes du modele metier.
- `functional-overview-medicalvision.puml` : vue fonctionnelle generale du parcours systeme.

## Export recommande

Ouvre les fichiers dans une extension PlantUML ou colle leur contenu dans PlantUML Online, puis exporte en SVG ou PNG haute resolution.

Pour un rapport, le format SVG est preferable si ton editeur l'accepte. Sinon, exporte en PNG avec une resolution elevee.

## Remarque

Le diagramme de classes est volontairement centre sur les entites metier persistantes. Il evite les details internes des composants React, schemas Pydantic et fonctions de service afin de rester lisible dans un document academique.
